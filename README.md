# Tema 2 – Fundamentos e Técnicas Avançadas

Trabalho individual  
Curso: Análise e Desenvolvimento de Sistemas  

## Descrição
Este projeto tem como objetivo demonstrar os fundamentos da programação
utilizando a linguagem Python, como entrada de dados, variáveis e estruturas
condicionais.

## Tecnologias utilizadas
- Python

## Como executar
1. Execute o arquivo `main.py`
2. Informe seu nome e idade
3. O programa exibirá o resultado na tela
